구현한 내용
 - 도형이 스플라인 곡선으로 날아온다.
 - 도형의 종류: 삼각형, 사각형, 오각형
 - 드래그 하여 도형을 슬라이스 한다: 마우스 드래그 경로가 도형의 변 2개와 교차할 경우
 - 잘린 도형을 작은 조각이 되어 아래로 떨어진다.
 - 도형을 클라이언트 밖으로 나가면 사라진다.
 - 3초마다 도형이 계속 날아온다.
 - 한번 슬라이스 한 도형을 다시 슬라이스 할 수 있다.

구현하지 못한 내용
 - 프로그램이 시작했을 때 날아오는 경로를 입력 받는 것


키보트 명령어
A키: 날아오는 속도 빠르게(1.5배까지)
S키: 날아오는 속도 느리게(0.5배까지)
Z키: 도형 그리기 모드 변경
X키: 도형이 날아오는 경로인 스플라인 곡선의 컨트롤 포인트 on/off
Q키: 프로그램 종료

기타 실행 시 알아야할 요소
 - 오각형 이상은 슬라이스 되지 않는다.
 - 오각형의 한 꼭짓점은 검은색이 되는 데 왜 그러는지 모르겠습니다
